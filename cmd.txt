# 6. 常用参数说明
# -----------------------------------------------------------------------------

# --policy_type: Policy 类型 (lerobot, custom)
# --policy_path: 模型路径 (LeRobot policy 必需)
# --dataset_root: 数据集路径 (LeRobot policy 必需，用于 metadata)
# --garment_type: 服装类型 (top_long, top_short, pant_long, pant_short, custom)
# --num_episodes: 每个 garment 的 episode 数量 (默认: 5)
# --max_steps: 每个 episode 的最大步数 (默认: 600)
# --enable_cameras: 启用相机渲染
# --device: 模拟设备 (cpu 必需，避免服装物理问题)
# --headless: 无 GUI 模式
# --save_video: 保存评估视频
# --video_dir: 视频保存目录
# --task_description: 任务描述 (VLA models 需要)
