# 四种服装类型数据一致性分析报告

## 1. 执行摘要

本报告分析了 LeHome Challenge 中四种服装类型 (pant_short, pant_long, top_long, top_short) 的模型成功率差异原因，通过 wandb 训练数据和评估日志分析，发现：

**关键发现**：
- **pant_short** (88.3% 成功率): 数据一致性最高，成功率最高
- **pant_long** (48.3% 成功率): 数据不一致导致成功率低
- **top_long** (73.3% 成功率): 数据一致性中等，成功率中等
- **top_short** (41.7% 成功率): 视觉特征不明显 + 数据不一致，成功率最低

## 3. 数据一致性分析

### 3.1 核心指标

| 数据集 | 成功率 | Corr CV | Mag CV | Grip CV | 内部 Std |
|--------|--------|---------|--------|---------|----------|
| **pant_short** | **88.3%** | **0.0199** | 0.0425 | **0.0265** | 0.0546 |
| pant_long | 48.3% | **0.0998** | 0.0488 | **0.0975** | 0.0489 |
| top_long | 73.3% | 0.0851 | **0.0853** | 0.0318 | 0.0462 |
| top_short | 41.7% | 0.0810 | 0.0258 | 0.0318 | 0.0462 |

**指标说明**：
- **Corr CV**: 协调性变异系数 - 衡量不同 garment 间左右手协调性的差异
- **Mag CV**: 动作幅度变异系数 - 衡量不同 garment 间动作速度的差异
- **Grip CV**: Gripper 操作变异系数 - 衡量不同 garment 间 gripper 使用频率的差异
- **内部 Std**: garment 内部 episodes 的协调性标准差

## 4. 失败模式分析

### 4.1 top_short 失败模式

评估条件 (阈值: [16, 27, 15, 29, 19]):
- p[0]-p[4] <= 16: 左右肩膀靠近
- p[2]-p[3] <= 27: 袖口靠近
- p[1]-p[5] <= 15: 下摆靠近
- p[0]-p[1] >= 29: 保持长度
- p[4]-p[5] >= 19: 保持长度

| 条件 | 失败率 | 解读 |
|------|--------|------|
| p[0]-p[4] (肩膀) | 82.5% | 肩膀对齐差 |
| **p[2]-p[3] (袖口)** | **90.5%** | **袖口对齐差 (主要问题)** |
| p[1]-p[5] (下摆) | 61.9% | 下摆对齐中等 |

**关键发现**: 90.5% 的失败是因为袖口没有对齐 → 短袖袖口视觉特征不明显

### 4.2 top_long 失败模式

| 条件 | 失败率 | 对比 top_short |
|------|--------|----------------|
| p[0]-p[4] (肩膀) | 90.4% | 相似 |
| **p[2]-p[3] (袖口)** | **63.5%** | **低 27%** |
| p[1]-p[5] (下摆) | 84.6% | 高 23% |

**关键发现**: 额袖的袖口对齐失败率低 → 长袖视觉特征明显

### 4.3 pant_long 失败模式
| 条件 | 失败率 | 解读 |
|------|--------|------|
| p[0]-p[4] (腰部) | 40.5% | 腰部对齐问题 |
| **p[2]-p[3] (裤脚)** | **0%** | **总是成功** |
| p[1]-p[5] (裆部) | 44.1% | 裆部对齐问题 |

**关键发现**: 裤脚对齐总是成功 → 裤子形状简单，主要失败在腰部和裆部

## 5. 左右手协调性分析

| 数据集 | 成功率 | 轨迹距离 | 左右手相关性 | 解读 |
|--------|--------|----------|--------------|------|
| pant_short | 88.3% | 18.33 | **0.73** | 高度协调 |
| pant_long | 48.3% | 28.90 | **0.40** | 协调性低 |
| top_long | 73.3% | 38.67 | **0.33** | 独立操作 |
| top_short | 41.7% | 37.50 | **0.31** | 独立操作 |

**发现**:
- pant_short 成功率高是因为双手协调操作 (相关性 0.73)
- top 类型需要双手独立操作 (相关性 ~0.32)
- pant_long 协调性低 (0.40) 但任务需要协调 → 失败

## 7. 根本原因总结

### 7.1 pant_short (88.3% 成功) ✓
- **数据一致性最高** (Corr CV = 0.0199, Grip CV = 0.0265)
- 无问题 Garments
- 任务简单 + 数据一致 = 成功率最高

### 7.2 top_long (73.3% 成功) ⚠
- 数据一致性中等
- **长袖视觉特征明显** → 袖口对齐失败率低 (63.5% vs 90.5%)
- 成功率中等

### 7.3 pant_long (48.3% 成功) ✗
- **协调性 CV 高** (0.0998, 是 pant_short 的 5 倍)
- **Gripper CV 高** (0.0975, 是 pant_short 的 3.6 倍)
- 80% 问题 Garments
- 不同 garment 操作方式差异大

### 7.4 top_short (41.7% 成功) ✗
- **短袖视觉特征不明显** → 袖口对齐失败率 90.5%
- 70% 问题 Garments
- 视觉识别困难 + 数据不一致

## 8. 改进建议

### 8.1 pant_long 改进方案
**问题**: 数据不一致 (不同 garment 操作方式差异大)

**方案 A: 增加训练数据量**
- 从 250 episodes 增加到 500+ episodes
- 增加每个 garment 的数据量

**方案 B: 数据增强**
- Gripper 时序噪声: 随机偏移 gripper 动作时机
- 速度扰动: 随机缩放动作幅度
- 时间缩放: 随机加速/减速动作序列

**方案 C: 模型改进**
- 增加训练步数: 20k → 50k+ steps
- 使用更强的正则化

### 8.2 top_short 改进方案
**问题**: 视觉特征不明显 + 部分数据不一致

**方案 A: 图像增强**
```yaml
image_transforms:
  crop_sleeve_region: true  # 裁剪袖口区域
  brightness: [0.8, 1.2]  # 增强亮度范围
  contrast: [0.8, 1.2]  # 增强对比度
  saturation: [0.5, 1.5]  # 增强饱和度
```

**方案 B: 多尺度训练**
```yaml
multi_scale: [0.8, 1.0, 1.2]
random_crop: true
```

**方案 C: 增加训练数据量**
- 录制更多短袖数据
- 确保袖口区域清晰可见

### 8.3 通用改进方案
**方案: 多任务学习**
- 在 four_types 数据集上联合训练
- 共享通用操作技能，提高泛化能力
- 已有: smolvla_four_types 模型

## 9. 优先级建议

1. **pant_long**: 增加训练数据量 (最简单，预期效果最好)
2. **top_short**: 图像增强 + 多尺度训练 (针对视觉问题)
